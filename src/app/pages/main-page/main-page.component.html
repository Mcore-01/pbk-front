Подбор банковских карт

Наш сервис

Интернет-сервис по подбору банковских карт, исходя из данных о тратах пользователя. Основная идея проекта заключается в поиске наиболее подходящих банковских продуктов, рассчитанных индивидуально для каждого человека.
